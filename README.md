# 📰 Flutter News App

A modern and responsive Flutter news application that fetches real-time news using API integration. The app features breaking news sliders, category-based news, search functionality, detailed news screens, and a clean modern UI.

---

# ✨ Features

- 🔥 Breaking News Carousel Slider
- 🔎 Search News Functionality
- 📰 Real-Time News API Integration
- 📱 Fully Responsive UI
- 🖼️ Network Image Handling
- ⚠️ Image Error Handling
- 🧩 Reusable Widgets
- 📄 Detailed News Screen
- 🎨 Modern UI Design

# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/itzlucky9/News_India_newsapp.git
```

---

## 2️⃣ Install Dependencies

```bash
flutter pub get
```

---

## 3️⃣ Run App

```bash
flutter run
```
